# 📊 KPI de Controle de Recebimento de Cargas

Este projeto apresenta um modelo de controle logístico focado em inspeção, recebimento e descarregamento de cargas.

## 🎯 Objetivo
Monitorar a qualidade e eficiência do processo logístico através de indicadores de performance (KPIs).

## 📈 Indicadores utilizados
- Taxa de avaria (%)
- Tempo médio de descarga
- Taxa de inspeção (%)
- Volume total recebido

## 🧠 Aplicação prática
Com esses indicadores, é possível:
- Identificar falhas no transporte ou manuseio
- Avaliar desempenho de fornecedores e transportadoras
- Reduzir avarias e retrabalho
- Melhorar o tempo de operação

## 🛠️ Ferramentas
- Excel (base de dados)
- Power BI (visualização e análise)
